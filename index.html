Dapplets

Dapplets is a protocol for permissionless, decentralized and unstoppable Augmented Web.

You need to install the browser extension to use Dapplets on other websites

Specification

As a crypto-economic standard, Dapplets define both data interfaces and incentivized roles.

On the data level it describes the life-cycle of the augmentation widget in the hosting site and the augmentation discovery protocol

Community incentives ensure audit and security services for the whole platform.

Applications

Dapplets protocol enables creation of unstoppable augmentations for third party websites. Here are few ideas:

  • Identity services: messaging & security alerts.
  • Scam and fake news alerts.
  • Payments and other crypto-economic addons for existing web communities.

Implementation

Community

There are different roles in Dapplets community:

  1. CoreDevs are working on the protocol and the platform.
  2. ModuleDevs are implementing modules and website adapters.
  3. AppDevs are implementing own business models.
  4. Auditors verify modules created by developers and signal an audit status.
