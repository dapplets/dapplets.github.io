Dapplets

Dapplets is a protocol for permissionless, decentralized and privacy-aware Augmented Web.

Specification

As a crypto-economic standard, Dapplets define both data interfaces and incentivized roles.

On the data level it describes the life-cycle of the augmentation widget in the hosting site and the augmentation discovery protocol

Community incentives create social structures providing audit and security for the whole platform.

Applications

Dapplets protocol enables creation of unstoppable augmentations for third party websites. Here are few ideas:

  1. Identity services: security alerts, ratings and feedbacks.
  2. Scam and fake news alerts.
  3. Payments and other crypto-economic addons for existing web communities.
Do you have more ideas for Dapplet Apps? Join our community and create your own business case!

Implementation

Dapplets is currently implemented as a browser plugin and as a javascript library (as used here). Out goal is to implement it in the browser natively.

Community

There are different roles in the Dapplets Project:

  1. CoreDevs are working on the protocol and the platform.
  2. ModuleDevs are implementing modules and website adapters.
  3. AppDevs are implementing own business models.
  4. Auditors verify modules created by developers and signal an audit status.
