Article: Secured Open Wallet Architecture (SOWA)

26.02.2020

The S.O.W.A. — Secured Open Wallet Architecture is a spin-off from the Dapplets project. It defines a general chain agnostic standard for scripts running in the Wallet’s sandbox and describing both the confirmation View and the Action execution plan.
